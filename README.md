# Water Quality Monitoring using Data Mining

## Project Overview
This project predicts water quality using Data Mining techniques and a Decision Tree Classification model.

## Dataset Features
- pH
- Turbidity
- Dissolved Oxygen
- Conductivity
- Temperature

## Algorithm Used
- Decision Tree Classifier

## Tools & Technologies
- Python
- Pandas
- Scikit-Learn
- Google Colab
- Git & GitHub

## Results
The model classifies water as Safe or Unsafe based on water quality parameters and achieves high prediction accuracy.
